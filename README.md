# Презентация QA Engineer Артема

Современный одностраничный сайт-презентация с интерактивными элементами и анимациями.

## Особенности

- ✅ One-page scroll с плавной навигацией
- ✅ Пошаговое раскрытие текста с кнопкой "Далее"
- ✅ Подсветка активных абзацев с анимацией glow
- ✅ Параллакс эффект для фоновых элементов
- ✅ Всплывающие иконки инструментов
- ✅ Подсветка ключевых слов (Node.js, Python Playwright, Agile и др.)
- ✅ Раскрывающиеся кейсы с анимацией
- ✅ Современный tech-style дизайн
- ✅ Адаптивная верстка

## Использование

1. Откройте `index.html` в браузере
2. Используйте навигацию вверху для быстрого перехода к разделам
3. Нажимайте кнопку "Далее" или используйте стрелки клавиатуры (↓/→) для раскрытия текста
4. Наводите курсор на заголовки кейсов для просмотра анонса
5. Кликайте на заголовки кейсов для полного раскрытия

## Структура

- **Hero** - Главный экран с приветствием
- **Skills** - Навыки и инструменты
- **Experience** - Опыт работы (3 компании)
- **What I look for** - Что ищу в работе
- **Why choose me** - Почему стоит выбрать
- **Cases** - Кейсы и истории успеха (5 кейсов)
- **Final** - Финальный блок с контактами

## Технологии

- HTML5
- CSS3 (с анимациями и градиентами)
- Vanilla JavaScript (без зависимостей)

## Настройка контактов

Отредактируйте ссылки в финальном блоке (строки 357-359 в `index.html`):
- Email: замените `artem@example.com` на реальный email
- LinkedIn: замените `https://linkedin.com` на ваш профиль
- Telegram: замените `https://t.me` на ваш Telegram

## Браузеры

Сайт протестирован и работает в:
- Chrome/Edge (рекомендуется)
- Firefox
- Safari
